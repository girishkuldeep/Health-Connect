# CampusMed+ | College Medical Assistance Website

A secure, institutional web platform designed to digitize healthcare delivery, patient records, and appointment scheduling at the IIT Ropar Medical Center. This project transitions traditional manual processes into a streamlined, privacy-conscious digital system.

🌐 **Live Demo:** [https://vercel.app](https://vercel.app)

## 🎯 Project Overview
* **Problem:** The existing campus medical center relied heavily on handwritten logs, leading to long waiting times, manual entry errors, and self-medication trends among students.
* **Solution:** Developed a web-based portal mapping user-specific interfaces for Students, Doctors, Pharmacists, and Administrators to handle end-to-end medical administration securely.
* **Privacy Focus:** Engineered specifically to bypass public cloud dependencies, keeping sensitive medical records within restricted or localized network environments to maintain institutional compliance.

## ⚙️ Key Technical Features & Workflows
* **Student Dashboard:** Features real-time appointment booking, personalized scheduling, and digital prescription access.
* **Doctor Portal (FIFO Queue):** Integrates an automated First-In, First-Out (FIFO) consultation triage, monthly availability management, and an interactive digital prescription-writing interface.
* **Pharmacy Management:** Handles inventory tracking (stock additions/deletions), pending prescription pickups via unique RX codes, and historical dispensing logs.
* **Administrative Controls:** Centralized system dashboard featuring real-time appointment feeds, user directories, role assignment changes, and medical staff authorization gates.

## 🛠️ Tools & Technologies Used
* **Frontend:** React.js / Next.js (User Interfaces & Dashboards)
* **Backend & Database:** Server-side data processing and localized database architecture for secure patient history retrieval
* **Deployment Platform:** Vercel
